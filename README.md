# Pos App


## Use By
- wails
- vue 
- Tailwindcss


## Develop
```
wails dev
```

## Build App

### build platform
```
wails build
```

### windows platform
```
wails build -platform=windows/arm64 # or windows/amd64
```

### mac platform
```
wails build -platform=darwin/arm64 # or darwin/amd64
```

### linux platform
```
wails build -platform=linux/arm64  # or linux/amd64
```
